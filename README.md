# Huntd Testing Project

QA practice project for testing the Huntd web and mobile applications.

## Project Scope
- Web application testing
- Mobile application testing
- Test documentation
- Test design
- Bug reporting
- Test execution and reporting

## Repository Structure

- `/test-plan` — test plan documentation
- `/decomposition` — feature decomposition
- `/permission-table` — permission testing matrix
- `/rtm` — requirements traceability matrix
- `/reports` — test summary reports
- `/assets` — images and supporting files

## Tools Used
- Jira
- TestRail
- GitHub
- VSCode

## Tested Platforms
- Huntd Web Application
- Huntd Mobile Application (MVP)
